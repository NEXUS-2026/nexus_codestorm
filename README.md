# Nexus Codestorm

## Structure

```
├── Backend/   # Python backend
└── Frontend/  # React + Vite frontend
```

## Getting Started

### Frontend
```bash
cd Frontend
npm install
npm run dev
```

### Backend
```bash
cd Backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```
